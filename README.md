Matéria
A disciplina de Introdução à Computação visa fornecer a base conceitual e prática indispensável para qualquer engenheiro de software. Ela abrange desde os fundamentos históricos e a arquitetura básica de computadores até a infraestrutura moderna de redes, internet e os modelos de governança de TI. O objetivo é compreender como o ecossistema tecnológico funciona a nível de hardware, comunicação e gestão estratégica para sustentar o desenvolvimento de sistemas complexos.
